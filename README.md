# Backtren SuperSimple

## Swagger

La documentación Swagger quedó habilitada para este proyecto.

### Acceder a la UI

Abre en tu navegador:

- http://localhost:3000/docs

### JSON OpenAPI

- http://localhost:3000/docs.json

### Instalación de dependencias

Si aún no instalaste las nuevas dependencias, ejecuta:

```bash
npm install
```

### Ejecutar la API

```bash
npm start
```

O con recarga automática:

```bash
npm run forest
```
